# selectMultiCheckbox

`selectMultiCheckbox` は、Webページ上で選択した範囲内にあるチェックボックスのチェック状態を反転できるブックマークレットです。

## できること

- テキストや要素をドラッグして選択した範囲内のチェックボックスを探す
- 選択範囲に含まれるチェックボックスのチェック状態をすべて反転する
  - チェックされていなければチェックし、チェックされていればチェックを外す
- `change` イベントが発生するので、ページ側の挙動にも対応しやすい

## 使い方（初心者向け）

### 1. ブックマークレットを作る

1. ブラウザのブックマークバーに新しいブックマークを追加します。
2. 名前は `複数チェックボックス選択` などわかりやすいものにします。
3. URL（またはロケーション）には、次のコードをコピーして貼り付けます。

```javascript
javascript:(function(){const s=window.getSelection();if(!s.rangeCount)return;const r=s.getRangeAt(0);document.querySelectorAll('input[type="checkbox"]').forEach(cb=>{if(r.intersectsNode(cb)){cb.checked=!cb.checked;cb.dispatchEvent(new Event('change',{bubbles:true}));}});})();
```

> ブックマークレットとは、ブックマークに保存した小さな JavaScript コードを実行する仕組みです。

### 2. チェックしたい範囲を選択する

1. 対象のページを開きます。
2. マウスでチェックボックスの近くをドラッグして、範囲選択します。
3. 範囲に含まれるチェックボックス全てをまとめて選択したいときに使います。

### 3. ブックマークレットを実行する

1. 先ほど作成した `複数チェックボックス選択` ブックマークをクリックします。
2. 選択範囲内にあるチェックボックスのチェック状態がトグル（反転）されます。
3. ページに `onchange` や `change` イベントが登録されている場合は、その処理も起動します。

## 使い方の例

- アンケートの複数選択項目
- メール一覧の複数チェックボックスを一度に選ぶとき
- タスク一覧の複数項目をまとめてチェックする場合

## 注意事項

- このブックマークレットは、`<input type="checkbox">` の要素を対象にしています。
- ページ内のすべてのチェックボックスではなく、ユーザーが選択した範囲内にあるものだけが対象です。
- ページ側でチェックボックスの状態を独自に管理している場合は、表示が更新されないことがあります。
- 一部の Web サイトでは、セキュリティ対策でブックマークレットの実行が制限される場合があります。
- JavaScript が無効になっているページでは動作しません。

## ファイル構成

- `script.js` : 読みやすくコメント付きの JavaScript 本体
- `script_oneliner.js` : ブックマークレット用に1行にまとめた JavaScript
- `test.html` : 動作確認用のチェックボックステストページ
- `README.md` : この説明書

## 動作確認用ページの使い方

1. `test.html` をダウンロードして、ブラウザで開きます。
2. ページ上のチェックボックスの近くをマウスでドラッグして選択範囲を作ります。
3. ブックマークバーの `複数チェックボックス選択` ブックマークレットをクリックします。
4. 選択範囲内にあるチェックボックスが自動でチェックされます。

> `test.html` は、ブックマークレットの動作確認を簡単に行えるように作成されています。
